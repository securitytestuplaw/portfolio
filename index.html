QA Case Studies

Detailed breakdowns of my testing process, from planning to bug reporting.

WEB TESTING

E-Commerce Checkout Flow

Comprehensive functional and regression testing of a multi-step checkout process for a high-traffic retail platform.

Tools: JIRA, Chrome DevTools, Excel

Test Cases: 45 Created

Bugs Found: 8 Critical

Test Scenarios & Execution

Positive Testing
  • User can add items to cart and update quantities.
  • Guest checkout functionality works.
  • Payment gateway processes valid cards (Stripe Sandbox).
Negative Testing
  • Checkout with empty mandatory fields.
  • Applying expired or invalid coupon codes.
  • Session timeout handling during payment.

Key Defect Discovered

Bug #EC-102: Total Calculation Error

Applying a percentage discount coupon resulted in negative tax calculation for specific regions, blocking payment completion.

Tools: Postman, JSON

Endpoints Tested: 12
